# Projeto Formulário de Matrícula

Projeto de formulário de matrícula para a escola de educação infantil "Estrelas do Amanhã".

## Sobre

Este projeto contém uma página estática em HTML e CSS que apresenta um formulário para cadastro de criança, informações de endereço, responsável e opções de matrícula.

## Estrutura do projeto

- `index.html` - página principal com o formulário.
- `styles/` - estilos CSS organizados por área e componentes.
- `assets/` - ícones e imagens usados na interface.

## Funcionalidades

- Campos de texto para nome, telefone, email e endereço.
- Seleção de sexo, turno de estudo e esporte preferido.
- Upload de arquivo para certidão de nascimento.
- Checkbox de concordância com termos e condições.
- Layout responsivo e visual inspirado em formulários modernos.

## Como usar

1. Abra o arquivo `index.html` em um navegador.
2. Preencha os campos do formulário.
3. Use o botão `Fazer matrícula` para simular o envio do formulário.

> Observação: este projeto é apenas front-end estático. Não há backend ou lógica de envio de formulário implementada.

## Tecnologias

- HTML
- CSS

## Contato

Projeto criado para demonstrar um formulário estilizado de matrícula escolar.
